I'm currently using ClickUp for task/project management on a personal level. It has a lot of great features but it also lacks in a few key areas. For example:

- **Buggy** - Right now updating a task due date doesn't always work. I reported it several days ago, I can work around it (some UIs are more likely to succeed than others) but this is basic functionality.
	- What makes it more frustrating is that there is a lot of complex functionality but the basic stuff seems to break sometimes. More focus on basics, less on bells and whistles please. :)
- **Crippleware** - Usually when I think of a free account one can use only the functionality that is truly free. I think of crippleware as being when one offers functionality but then disables it without paying for an upgrade.
	- ClickUp does this somewhat frequently (time tracking, custom field values) and I find this highly frustrating - especially since it isn't clear when one begins using the feature that it will be crippled later.
- **Pricing** - Its pretty average pricing but as with many other productivity suites one has to buy a whole bundle of functionality one doesn't need to get the few functions one does need.

I'll eventually move on to something else - unless ClickUp really picks up their game. But for the moment I'm making due. I've used so many productivity systems over the years and they all lack in various ways.

Someday, maybe, I'll build my own. :)

If you are using ClickUp, here are a few things I've found helpful:
# Get Away from the Personal List
By default ClickUp likes to throw things in a "Personal List" which operates according to funky rules I don't understand (nor do I care to). Items in it don't seem to be visible in the "Everything" view nor do they appear in the "Inbox". The way I found them was through the "Home" view which includes a "Personal List" widget.

Needless to say, a few tasks went into this black hole before I figured out what was going on. I recommend abandoning the Personal List.

Unfortunately, ClickUp really wants to throw new tasks into that list sometimes. You can setup an automation that moves items ClickUp throws in there into another list of your choice. Hover over the Personal List and you should see a lightning bolt icon, clicking on it opens up the Automations for that list. You can create an automation that moves items from that list as they are created to another list of your choice.

You'll need to move the existing items over yourself.
# Use Checklists instead of Tasks
Use Tasks, but especially for recurring tasks, use checklists. It can quickly get overwhelming trying to sift through hundreds or thousands of tasks.

So, for example, I have a task that contains ideas for things to do with my son. Previously I had a task with sub-tasks - but the sub-tasks crowd the UI (and sure, you can not show them, but sometimes I want to show the sub-tasks!).
# Use Lists Instead of Folders
I started out with lots of folders with lists inside of them. I subdivided the lists - e.g. I have recurring lists that cover various types of responsibilities - groups I lead, health, chores, and so on.

Sometimes these are good, but if there are only going to be a few items on the list you can often use a single list without nesting inside a folder and reduce the clickyness.
